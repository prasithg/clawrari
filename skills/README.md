# Skills

Curated skill recommendations for a Clawrari setup.

## Recommended Skills

These are the skills that power the reference setup. Install via ClawHub:

### Communication
- **slkcli** — Slack integration (read/write channels, threads, reactions)
- **gog** — Google Workspace (Calendar, Gmail, Drive, Docs, Sheets)

### Development
- **gh** — GitHub CLI (repos, PRs, issues)
- **codex** — OpenAI Codex CLI for coding tasks
- **claude-code** — Anthropic Claude Code for coding tasks

### Research
- **web-search** — Perplexity Sonar for web search
- **web-fetch** — URL content extraction

### Content
- **x-api** — Twitter/X posting and reading (official API, OAuth 1.0a)
- **browser** — Browser automation for platforms without APIs

## Installation

```bash
# Install a skill from ClawHub
openclaw skill install <skill-name>
```
