Quick Start

# Clone it
git clone https://github.com/prasithg/clawrari.git
cd clawrari

# Run the bootstrap (2 minutes)
./bootstrap/init.sh

# Start OpenClaw
openclaw gateway start

The bootstrap asks a few questions, generates your workspace, and you're live.
